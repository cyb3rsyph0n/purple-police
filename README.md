# PurplePolice

A World of Warcraft addon that shows enchant rank icons on enchantable gear in the character screen.

## Installation

### From CurseForge
Download from [CurseForge](https://www.curseforge.com/wow/addons/purplepolice) or use a client like CurseForge App or WowUp.

### Manual Installation
1. Download the latest release
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or `/reload`

## Features
- Displays enchant rank icons on enchantable gear slots
- Helps identify gear that needs enchanting
